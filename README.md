# Installation

```bash
git clone git@github.com:nico-develop/TU-WMD03.git
cd TU-WMD03

npm i
```

# Excecuter les tests 

Mac/Linux

```bash
./node_modules/.bin/mocha
```

Windows

```bash
node node_modules/.bin/mocha
```
